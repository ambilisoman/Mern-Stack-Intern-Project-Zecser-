# Login Form UI – React (Day 5 Task)

##  Objective

Build a responsive Login Form using React to demonstrate state management, event handling, and validation.


##  Features

* Email & Password inputs
* Regex validation (real-world format)
* Show / Hide password (Font Awesome eye icon)
* Error handling & messages
* Submit handling
* Login success alert
* Redirect to Welcome page
* Responsive design (image hidden on small screens)
* Reusable Button component

---

##  Tech Used

* React (Vite)
* useState Hook
* React Router DOM
* CSS (Flexbox + Media Queries)
* Font Awesome CDN


##  Validation Rules

* **Email:** Standard email regex format
* **Password:** Min 6 chars, at least 1 letter & 1 number

##  Output Flow

Valid Login → Success Alert → Redirect → “Welcome to Zecpath”
