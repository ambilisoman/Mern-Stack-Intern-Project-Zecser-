# Job Listings Application

A React-based job listing application that fetches remote job data from the Remotive API.

## Features
- Browse remote jobs by category
- Search functionality
- Responsive design
- Job details page
- Clean and modern UI

## Tech Stack
- React 18
- React Router DOM
- Axios for API calls
- CSS for styling

## API Used
This application uses the [Remotive API](https://remotive.com/api-documentation) to fetch remote job listings.
