# Zecpath Job Portal

A **React-based remote job listing app** that fetches jobs from the public Remotive API. The app demonstrates **useEffect**, **component lifecycle**, **API fetching**, **loading & error states**, and **responsive design**.

---

## Features

* Fetches **remote jobs** dynamically using the [Remotive API](https://remotive.com/api/remote-jobs).
* **Search jobs** by title or keyword in real-time.
* **Loader** displayed while fetching data.
* **Error handling** for failed API requests.
* **Auto-refresh** every 60 seconds to keep jobs up-to-date.
* **Responsive design**:

  * Sidebar for desktop
  * Bottom navigation for mobile
* **Reusable Button component** for consistency.


## Tech Stack

* **React** – Frontend library
* **JavaScript** – Logic & API handling
* **CSS** – Styling and responsiveness
* **Public API** – Remotive remote jobs API
