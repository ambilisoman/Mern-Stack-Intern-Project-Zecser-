# Job Portal React App

A simple job portal application built with React that fetches real job listings from the Remotive API.

## Features

-  Search for jobs by title or company
-  Responsive design with mobile-friendly interface
-  Job cards with detailed information
-  Apply Now button with confirmation alert
-  Clean and modern UI

## API Integration

The app uses the **Remotive API** to fetch real job listings:
- API Endpoint: `https://remotive.com/api/remote-jobs`
- Features real-time job search
- Returns job details including company, location, and requirements
